MongoDB::repair

(PECL mongo >=0.9.0)

MongoDB::repair — このデータベースを修復・圧縮する

説明

public array MongoDB::repair ([ bool $preserve_cloned_files = FALSE [, bool $backup_original_files = FALSE ]] )

パラメータ

preserve_cloned_files

修復に失敗した場合に、コピーしたファイルを保持します。

backup_original_files

元ファイルをバックアップするかどうか。

返り値

データベースの応答を返します。
