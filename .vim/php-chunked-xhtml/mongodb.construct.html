MongoDB::__construct

(PECL mongo >=0.9.0)

MongoDB::__construct — 新しいデータベースを作成する

説明

MongoDB::__construct ( Mongo $conn , string $name )

パラメータ

Mongo conn

データベース接続。

name

データベース名。

返り値

データベースを返します。

エラー / 例外

データベース名が無効な場合に InvalidArgumentException をスローします。
