MongoDB::execute

(PECL mongo >=0.9.3)

MongoDB::execute — JavaScript コードをデータベースサーバ上で実行する

説明

public array MongoDB::execute ( mixed $code [, array $args = array() ] )

パラメータ

code

実行したい MongoCode あるいは文字列。

args

code に渡す引数。

返り値

評価した結果を返します。
