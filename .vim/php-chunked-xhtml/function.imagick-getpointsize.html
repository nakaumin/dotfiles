Imagick::getPointSize — Gets point size

説明

string Imagick::getPointSize ( void )

Returns the objects point size property. このメソッドは、ImageMagick バージョン 6.3.7 以降で Imagick をコンパイルした場合に使用可能です。

返り値

Returns a float containing the point size.
