MongoGridFS::remove

(PECL mongo >=0.9.0)

MongoGridFS::remove — ファイルをコレクションから削除する

説明

public boolean MongoGridFS::remove ([ array $criteria = array() [, boolean $just_one = FALSE ]] )

パラメータ

query

探したいファイル名あるいは条件。

just_one

マッチしたファイルをひとつだけ削除する。

返り値

削除に成功したかどうかを返します。
