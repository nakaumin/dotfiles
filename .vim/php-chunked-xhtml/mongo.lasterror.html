Mongo::lastError

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

Mongo::lastError — 直近のデータベース操作でエラーが発生したかどうかを調べる [非推奨]

説明

public array Mongo::lastError ( void )

かわりに MongoDB::lastError() を使ってください。

パラメータ

この関数にはパラメータはありません。

返り値

エラーが存在する場合はエラーを、存在しない場合は NULL を返します。
