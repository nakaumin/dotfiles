インストール手順

この » PECL 拡張 モジュールは PHP にバンドルされていません。

これらの関数を使用するには、オプション --with-hyperwave[=DIR] を使用して Hyperwave サポートつきで PHP をコンパイルする必要があります。

Windows ユーザがこれらの関数を使用するには、php.ini の中で php_hyperwave.dll を有効にします。 この PECL 拡張モジュールの DLL は、現在存在しません。 Windows でのビルド も参照ください
