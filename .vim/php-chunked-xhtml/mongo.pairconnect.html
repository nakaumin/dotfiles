Mongo::pairConnect

(PECL mongo >=0.9.0)

Mongo::pairConnect — ペアのデータベースサーバに接続する

説明

public boolean Mongo::pairConnect ( void )

ペアの接続を作成するには、$this->server を "server1,server2" 形式の文字列にしなければなりません。

パラメータ

この関数にはパラメータはありません。

返り値

接続に成功したかどうかを返します。

エラー / 例外

データベースへの接続に失敗した場合に MongoConnectionException をスローします。
