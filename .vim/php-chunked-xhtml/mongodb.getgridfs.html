MongoDB::getGridFS

(PECL mongo >=0.9.0)

MongoDB::getGridFS — このデータベースに格納されているファイルを扱うためのツールキットを取得する

説明

public MongoGridFS MongoDB::getGridFS ([ string $arg1 = "fs" [, string $arg2 = NULL ]] )

パラメータ

arg1

引数がこれだけだった場合は、ファイルやチャンクのコレクション。 それ以外の場合はファイルコレクションの名前。

arg2

チャンクコレクションの名前。

返り値

このデータベースの新しい gridfs オブジェクトを返します。
