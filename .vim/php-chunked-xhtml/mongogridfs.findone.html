MongoGridFS::findOne

(PECL mongo >=0.9.0)

MongoGridFS::findOne — 条件にマッチする単一のファイルを返す

説明

public MongoGridFSFile MongoGridFS::findOne ([ mixed $query = array() ] )

パラメータ

query

探したいファイル名あるいは条件。

返り値

ファイル、あるいは NULL を返します。
