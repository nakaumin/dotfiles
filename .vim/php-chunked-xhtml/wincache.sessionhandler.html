WinCache Session Handler

The WinCache session handler (available since WinCache 1.1.0) can be used to configure PHP to store the session data in shared memory user cache. Using shared memory instead of the default file session storage helps improve performance of PHP applications that store large amount of data in session objects.

To configure PHP to use WinCache session handler set the php.ini setting session.save_handler to wincache.

例1 Enabling WinCache session handler

session.save_handler = wincache

注意: In beta release of WinCache Extension 1.1, the user cache is kept in memory and is not backed by a file system storage. This means that the session data stored in the shared memory will be lost when IIS application pool is recycled.
