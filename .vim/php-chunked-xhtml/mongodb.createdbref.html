MongoDB::createDBRef

(PECL mongo >=0.9.0)

MongoDB::createDBRef — データベース参照を作成する

説明

public array MongoDB::createDBRef ( mixed $ns , mixed $a )

パラメータ

ns

データベース参照が指すコレクション。

a

参照を作成するオブジェクトあるいは ID。

返り値

データベース参照配列を返します。
