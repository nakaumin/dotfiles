インストール手順

PECL からのインストール

  1. http://snaps.php.net/win32/ から php_win32service.dll がダウンロード可能です。 PHP のバージョンに一致する PECL_X_X フォルダを選択します。

  2. php_win32service.dll を extension_dir にコピーします。

  3. php.ini で拡張モジュールを読み込みます。 

    extension=php_win32service.dll
