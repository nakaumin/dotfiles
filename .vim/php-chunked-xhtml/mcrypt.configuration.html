実行時設定

php.ini の設定により動作が変化します。

Mcrypt 設定オプション
名前 デフォルト 変更の可否 変更履歴
mcrypt.algorithms_dir NULL PHP_INI_ALL PHP 4.0.2 以降で使用可能です。
mcrypt.modes_dir NULL PHP_INI_ALL PHP 4.0.2 以降で使用可能です。
PHP_INI_* モードの詳細および定義については どこで設定を行うのか を参照してください。
