Mongo::forceError()

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

— データベースエラーを作成する [非推奨]

説明

public bool Mongo::forceError ( void )

かわりに MongoDB::forceError() を使ってください。

パラメータ

この関数にはパラメータはありません。

返り値

データベースの応答を返します。
